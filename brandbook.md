# Брендбук: INST НА МІЛЬЙОН (Bright & Structured)

## 1. Візуальна концепція та філософія (Moodboard)
**Естетика:** "Smart Educational & Bright" (Яскравий освітній стиль).
Дизайн сайту має бути максимально чистим, структурованим та зрозумілим. Він викликає довіру, виглядає сучасно та експертно. Головна фішка — використання білого простору, великих об'ємних тіней для карток та яскравих кольорових акцентів для привернення уваги до ключової інформації.

## 2. Кольорова палітра

| Назва | Hex/RGBA | Застосування |
| :--- | :--- | :--- |
| **Clean White (Чистий Білий)** | `#FFFFFF` | Фони карток та основних блоків. |
| **Light Gray (Світло-сірий)** | `#F8F9FA` | Тло сайту, яке дозволяє білим карткам виділятися завдяки тіням. |
| **Expert Yellow (Експертний Жовтий)** | `#FBB124` | Головний акцент. Використовується для виділення номерів, бейджів, заголовків та кнопок. |
| **Alert Red (Сигнальний Червоний)** | `#E63946` | Додатковий акцент для важливих маркерів (наприклад, слово "ТЕМА:"). |
| **Dark Charcoal (Темно-графітовий)** | `#1E1E24` | Основний текст. Забезпечує ідеальний контраст і не "ріже" очі як чистий чорний. |

## 3. Типографіка

Шрифтова пара має бути дуже простою, геометричною та читабельною.

*   **Заголовки (H1, H2, H3):** *Montserrat* (Google Fonts)
    *   **Стиль:** Bold (700) та ExtraBold (800).
    *   **Застосування:** Надає дизайну ваги та впевненості. Заголовки масивні та "гучні".
*   **Основний текст (Body):** *Inter* (Google Fonts)
    *   **Стиль:** Regular (400) та Medium (500).
    *   **Застосування:** Ідеально читається на будь-яких екранах, використовується для списків та описів.

## 4. UI-Елементи та Стилістика

### 4.1. Картковий дизайн (Card UI)
Вся інформація подається у вигляді об'ємних білих карток на світло-сірому фоні сайту.
*   `background: #FFFFFF;`
*   `border-radius: 20px;`
*   `box-shadow: 0 16px 40px rgba(0, 0, 0, 0.08);`

### 4.2. Блоки "Урок / Крок"
Яскрава деталь, що структурує інформацію (як на референсі):
*   Жовтий блок із заокругленими кутами, білий жирний текст номера ("01"), під ним слово "КРОК" або "УРОК".

### 4.3. Акценти в тексті
Використання червоного кольору для маркерів:
*   `<span style="color: #E63946; font-weight: bold;">ТЕМА:</span>`

### 4.4. Кнопки
Кнопки масивні, з легким градієнтом або чистим жовтим кольором, заокруглені, з м'якою тінню для ефекту натискання.

## 5. Загальне враження
Дизайн не повинен відволікати. Він працює як ідеально складена презентація топового спікера: чітко, яскраво, структуровано. Погляд користувача сам має ковзати від номера кроку до його опису.
