# Архитектура и История проекта (Логика)

## Цель файла
Этот документ создан для того, чтобы даже через 10 лет было понятно, почему и как был создан этот проект, какие технологии использовались и какие архитектурные решения были приняты.

---

## 1. Концепция и Задумка
**Проект:** Продающий лендинг "INST НА МІЛЬЙОН"
**Модель бизнеса:** "Done With You" — мы берем на себя сложную работу (сценарии, монтаж, таргет), клиент только снимает видео по нашим сценариям.
**Целевая аудитория:** Девушки beauty-мастера (маникюр, брови, волосы, макияж).
**Целевое действие:** Бесплатный 10-минутный звонок (клиент выбирает: телефон или Zoom).
**Язык контента:** Украинский.

### Три тарифных пакета:
- **Тест ($49):** 5 видео, базовый формат — для пробы
- **Стандарт ($99):** 10 видео, полная команда (сценарист, монтажер, продюсер, таргетолог)
- **Про ($249):** 15 видео + ИИ аватар + $100 рекламного бюджета

### Ключевая идея:
Мы платим узкопрофильным специалистам, а клиент получает доступ к работе с целой командой по цене, которая значительно ниже, чем нанимать этих людей по отдельности.

## 2. Выбор Технологий (Стек)
Мы используем "фундаментальный проверенный стек", чтобы сайт был максимально быстрым, надежным и не зависел от устаревающих сложных фреймворков.

*   **Структура:** Semantic HTML5 (теги `<header>`, `<section>`, `<footer>` для правильного SEO).
*   **Стилизация:** Vanilla CSS с CSS Custom Properties (`:root`) для легкой смены тем и цветов. Без библиотек.
*   **Логика:** Vanilla JavaScript (ES6+) — IntersectionObserver для анимаций, аккордеон FAQ, модальные окна.
*   **Аналитика:** Meta Pixel (fbq) для ретаргетинга, собственный tracker.js для детального отслеживания поведения (секции, скролл, клики, время, выход).
*   **Шрифты:** Google Fonts — Cormorant Garamond (заголовки) + Inter (текст).

## 4. Структура лендинга (Секции по порядку)
```
1. Header (фиксированный, blur-фон, лого + навигация + CTA)
2. Hero (ценностное предложение, "Done With You", CTA-кнопка)
3. "Це для тебе, якщо ти..." (4 карточки с болями ЦА)
4. "Як це працює" (4 шага: дзвінок → сценарії → зйомка → клієнти)
5. "Що входить" (6 карточек: сценарист, монтажер, продюсер, таргетолог, ІІ аватар, бюджет)
6. "Тарифи" (3 карточки: Тест/Стандарт/Про)
7. "Що ти отримаєш" (4 карточки результатов)
8. FAQ (6 вопросов, аккордеон)
9. Финальный CTA (изображение + текст + кнопка)
10. Footer
+ Модальное окно бронирования (имя, телефон, выбор телефон/Zoom)
+ Плавающая CTA-кнопка на мобильном
```

## 5. Дизайн-решения
- **Палитра:** Жемчужно-белый фон, пудровые акцентные блоки, тёплый бежевый/розовое золото для CTA (подробнее в brandbook.md)
- **Шрифты:** Cormorant Garamond для элегантных заголовков с засечками (beauty/fashion эстетика), Inter для идеальной читабельности текста
- **Анимации:** Плавное появление элементов при скролле (IntersectionObserver), мягкие hover-эффекты, pulse на hero-изображении
- **Мобильная версия:** Mobile-first responsive, плавающая CTA-кнопка внизу экрана, бургер-меню

## 6. Аналитика — Что мы отслеживаем
| Событие | Описание | Файл |
|---|---|---|
| `page_load` | Загрузка страницы (referrer, размер экрана) | tracker.js |
| `section_view` | Пользователь доскроллил до секции (какая именно, порядок) | tracker.js |
| `scroll_depth` | Глубина скролла (25%, 50%, 75%, 90%, 100%) | tracker.js |
| `cta_click` | Клик по любой CTA-кнопке (текст кнопки, секция) | tracker.js |
| `time_on_page` | Время на странице (10/30/60/120/300 секунд) | tracker.js |
| `modal_open` | Открытие модального окна бронирования (какой пакет) | main.js |
| `booking_submit` | Отправка формы бронирования (имя, телефон, метод, пакет) | main.js |
| `Lead` (Meta Pixel) | Конверсия — отправка формы | main.js |
