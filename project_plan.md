# План проекта: Инстаграмм на миллион

## Текущий статус
**Стадия:** Разработка (Этап 3) — основной код реализован.

## Описание проекта
Создание продающего одностраничного лендинга для продукта "Инстаграмм на миллион" — сервиса формата **"Done With You"** (помощь в ведении соцсетей для beauty-мастеров). Команда берет на себя сценарии, монтаж, продюсирование и таргетированную рекламу.

- **Язык:** Украинский
- **Стек:** HTML5 + CSS3 + Vanilla JavaScript (фундаментальный)
- **Целевая аудитория:** Девушки б'юті-мастера
- **Целевое действие:** 10-минутный бесплатный звонок (телефон или Zoom, на выбор клиента)
